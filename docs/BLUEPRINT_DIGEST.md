## Product Bet

The blueprint argues against a thin Argaam-style copy. The useful product is a UAE public-markets intelligence app that combines:

- Official exchange data
- Issuer disclosures and IR material
- Local business news
- Global macro and commodity signals
- Bilingual Arabic/English document workflows
- Explainable AI research support

## Build Choice

The document's embedded Codex prompt asks for Next.js, PostgreSQL, Prisma, Docker, queues, and OpenAI. For this isolated build, I adapted the product vision to Khalid's ecosystem instead:

- Static PWA
- Deterministic Python brain
- JSON data contract
- GitHub Pages / GitHub-as-DB ready
- Free/mock-first, provider-swap-ready

## MVP Implemented Here

- Market pulse
- Watchlist cards
- Event strip
- Official disclosure timeline with Arabic originals
- Stock page tabs
- Global factor exposure map
- Deterministic scoring
- AI Research Support tab with evidence references
- Alerts flow
- Screeners
- Admin source diagnostics and launch-readiness blockers

## Explicit Non-Goals

- No brokerage execution
- No personalized financial advice
- No unlicensed real-time redistribution
- No scraped-live posture in the isolated build
